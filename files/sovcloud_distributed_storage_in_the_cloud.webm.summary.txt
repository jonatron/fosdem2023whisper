The talk given at the FOSDEM conference in 2023 focused on distributed storage in the cloud. The speaker provided a general overview of the options available, discussing the difference between locked-in cloud providers and open source solutions. They emphasized the importance of considering factors such as deployment flexibility, vendor support, and the ability to contribute to the software. The speaker also discussed various storage types, including node local storage, network block storage, file storage, object storage, and databases. They highlighted both proprietary and open source solutions in each category and discussed the complexities and differences within the database space. Overall, the talk aimed to provide an understanding of the options available for distributed storage in the cloud, while acknowledging that there is no one-size-fits-all solution and encouraging the audience to ask questions and challenge the information presented.
