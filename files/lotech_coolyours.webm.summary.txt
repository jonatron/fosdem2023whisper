

In this talk, Pedro Silva discussed how online collaboration can be personalized via the use of custom buttons, XML files, post message API, and variables. He discussed the different features, like content controls and accessibility check, that can be used to customize the software to fit different workflows. He also discussed language translation and the cool features that can be added by hacking directly on the interface. Finally, he discussed the advantages of taking part in the open source collaboration and how to use the weblate and matrix chat for assistance.
