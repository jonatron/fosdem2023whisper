

This talk discussed the capabilities of the Raspberry Pi Zero, a 15 euro full Linux PC, and the various ways it can be used in open source software development. It highlighted the Raspberry Pi Locator service which makes it easier to find and purchase the raspberry pi board, a project called Py4J which allows developers to connect and control electronic components, as well as various articles, tools and libraries such as Azure Zulu, CropiOS, FXGL, Vaadin, and G-Bang. It concluded with a demonstration of using Kotlin on the raspberry pie to control the GPIOs.
