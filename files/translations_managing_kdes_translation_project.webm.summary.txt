In this talk from the FOSDEM conference in 2023, the speaker discusses the KDE Translations project, which focuses on translating the user interface, documentation, web pages, and other elements of KDE software into various languages. They explain the structure of the project, including the global coordination team and individual language teams, as well as the workflow for translating different types of files. The speaker also highlights the large number of strings that need to be translated and the challenges of managing translations branches and scripts. They discuss future goals, such as increasing the number of active teams and improving the web interface. Overall, the talk provides insights into the complexities and processes involved in translating open source software.
