 

This talk discussed different tools that can be used to manage dependencies in free and open source software development. Some of the tools covered included Maven, Gradle, IntelliJ IDEA, Dependable, Renovate and Sneak. The tools provide features such as editing, scanning and upgrading dependencies, alerting on vulnerabilities, configuration, dashboards and PR bots. The talk also overviewed migration tools such as Refaster and OpenRee.
