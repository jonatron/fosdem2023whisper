The speaker gives an update on the work done on content controls in Write or in the past year. Content controls refer to rich text content controls, which are similar to input fields, but with the ability to have rich text and multiple text portions. The talk covers various improvements, including adding aliases or tags to content controls to provide helpful context information, enabling them to be used in read-only documents, supporting random access to content controls, and better handling of right-to-left languages. The speaker emphasizes that the feature set of content controls in Write or should be fully compatible with the feature set in Word.
