

This lighting talk discussed the Wayland display protocol, which was started over 15 years ago with the goal of replacing X11, the Yocto project which is a collaborative project of the Linux Foundation for building a custom embedded Linux distribution, and the Remote Desktop Protocol (RDP) which can be used as an alternative to VNC for remote screen sharing. It gave an example of how to configure Weston init file with RDP support, as well as some tips on using Yocto project, and Open Embedded build framework.
