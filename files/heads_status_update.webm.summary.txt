The talk at the FOSDEM conference in 2023 was given by Cyril Arion, the main Ed's maintainer. Ed's is an open-source firmware that serves as a built system and runtime environment based on coreboot and Linux. Its goal is to facilitate accessibility to security and privacy for non-tech-savvy individuals by reducing complexity and providing customizable policies. Ed's utilizes TPM for secure boot and supports features such as verified boot, detached signature verification, and sealed secrets. It also allows for external and internal flashing of ROMs. The talk covered various updates and upcoming features, such as maximized ROMs, support for TPM 2.0, cleanroom GPT key generation, write protection, and international keyboard support. Questions from the audience included support for Talos II platform and remote attestation.
