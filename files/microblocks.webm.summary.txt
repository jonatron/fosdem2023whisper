The speaker demonstrated the features of MicroBlocks, a free and open-source programming environment for physical computing. They showed that MicroBlocks is capable of displaying data from sensors, handling multitasking through its parallel processing capabilities, and is portable as it can be used with many different types of boards. The speaker also talked about its shareability features and how code can be shared through different methods. Lastly, the speaker demonstrated how external sensors can be connected to a microcontroller to create interactive projects. The talk also included a Q&A section.
