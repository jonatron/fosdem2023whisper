

In this talk, Daniel Esteban discussed visual programming language and how it can be used to create software with Go. Specifically, he showcased Blockly, a JavaScript library compatible with all major browsers, and how it can be used to create code graphically. He provided examples such as a "Hello Foster" script, a robotic fish tank temperature monitor, and a website visit counter. He also discussed the limitations of Blockly and how to work around them.
