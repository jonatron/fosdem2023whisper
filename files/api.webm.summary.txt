In this talk, the speaker, Bobor, discusses the importance of API observability and how it can be achieved using API gateway plugins. He introduces himself and talks about his interest in API observability. He explains the concept of APIs and how they are essential in today's API-centric world. Bobor then highlights the role of API gateways as a central point for observing APIs. He explains how API gateway plugins can be used to enable the three pillars of API observability: logging, tracing, and metrics. He mentions the availability of multiple API gateway plugins and how they can be customized to fit specific needs. Bobor provides a live demo of how to enable observability using Prometheus plugin in API 6. He concludes by mentioning additional resources and opens the floor for questions.
