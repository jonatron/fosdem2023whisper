 

In this talk, Balazs Vargo discussed how WebAssembly, M-Scripten technology, and headless conversion were used to develop the Colabora online Office. He explained the process of porting C and C++ code to WebAssembly and the various components needed to do so, including the M-Scripten compiler toolchain. He also discussed how the document web page can be modified for the future and how the Unia API and C++ code can be used from JavaScript.
