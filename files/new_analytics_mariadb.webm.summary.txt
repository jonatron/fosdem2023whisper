The speaker from MariaDB Corporation discusses new and upcoming features in MariaDB column store, an all-lib storage engine for MariaDB server. These features include filtering vectorization, external group buy, LZ4 compression for data files, extent map scalability improvement, a merge of primary and ex-manager processes, full TPCH support, an external distinct process, and Order By Rewrite, among others. The talk also touches on the MariaDB column store version numbering system.
