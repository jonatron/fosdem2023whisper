The talk is about building a plant monitoring app using InfluxDB, Python, and Flask with Edge to Cloud Replication. The speaker, Zoe Steinkamp, explains how to set up IoT hardware, use tools like Flask and InfluxDB, and ingest data into the database. She also covers querying data with Flux and SQL, setting up Edge data replication, and visualizing the data using Plotly. The talk provides step-by-step instructions and code examples for building the app, and includes a link to the project's GitHub repository.
