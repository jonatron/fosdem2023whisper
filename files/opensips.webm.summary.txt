 

This talk summarizes the features of OpenSIPs 3.3, a microservices platform that caters to various components of communication technology like IM, RCS, IMS, UC, call center and more. In particular, OpenSIPs 3.3 provides support for the MSRP protocol for features like RCS and parallel work for call center agents. It also improves verbosity with regards to how its internal state looks like and provides control of TCP connections.
