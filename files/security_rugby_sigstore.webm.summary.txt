

In this talk, James Strong and Luis Denham Patty explain how the open source platform Sixth Store is being used to improve the security of technology supply chains, while drawing comparisons to rugby in terms of the specialized positions and trust needed for a successful team. They also discussed various components of Sixth Store, such as trust routes, using Let's Encrypt for security certificates, OITC for end user identification, and Cosine for Glue tying it all together.
