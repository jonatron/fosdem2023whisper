The talk was about autoscaling feature in Kubernetes and how it can be done using CADA for RGW. The speaker discussed what CADA is and its basic concepts, including its ability to automatically scale Kubernetes resources, support for custom metrics, and its architecture. The speaker then focused on RGW and demonstrated how autoscaling works with CADA using a performance evaluation tool called HS Bench. The talk concluded with a Q&A session where the speaker answered questions about scaling down to zero, scaling OSDs, and suggested tuning.
