 

Olivier discussed a new user interface toolkit called Slint that is 100% implemented in Rust and meant for desktop and embedded use. Slint uses a domain-specific language that is as simple to learn as any other library's API. He demonstrated the live preview feature and running a demo on a Raspberry Pi Pico, showcasing its compatibility and performance. The project is open source under a GPL license and is set to have a 1.0 release in February of 2023. Developments for the future include drag and drop and design tools for even more user accessibility.
