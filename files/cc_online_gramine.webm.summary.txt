The talk covers an open-source project called Grameen, a library OS that emulates Linux kernel and runs unmodified guest in user space. The talk discusses technical details of Grameen, new features added, and future plans. The talk also covers EDMM, on-tly dynamic memory management, which was recently added. Grameen currently supports two backends, Linux and Linux SGX. The talk includes a discussion of EDMM workflow, the expensive deallocation flow, and ongoing work to optimize EDMM. The talk ends with information on how to contribute and get involved with Grameen.
